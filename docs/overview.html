jGet - java file downloader with bandwidth limit
jGet usage:
java -jar jget-<version>.jar <params>
<params> are:

  • -n number of working threads (1,2,3,4,...). Default is 2 working thread.
  • -l overall speed limit for all working threads in bytes per second (may use suffixes according IEC60027-2 ). Default is 10KiB.
  • -f input file name. File must contains URL with file name for saving through space. Line by line.
  • -o output folder. Default is current folder.
